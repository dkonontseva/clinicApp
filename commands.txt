Итератор (iterator) - это объект, который возвращает свои элементы по одному за раз.
С точки зрения Python - это любой объект, у которого есть метод__ next__.
Этот метод возвращает следующий элемент, если он есть, или возвращает исключение StopIteration,
когда элементы закончились. Кроме того, итератор запоминает, на каком объекте он остановился в последнюю итерацию.

Чтобы представить таблицы базы данных в виде Python-классов, используют декларативный класс.
В этом подходе каждый класс будет представлять таблицу в базе данных, а атрибуты класса — столбцы таблицы.
